You can find the documentation about the testsuite in [GPAC's wiki](https://wiki.gpac.io).

The testing documentation is organized around three use cases: 
 - [CI and Github](https://wiki.gpac.io/Build/tests/CI-and-Github/): info about GPAC's CI and testing features for github contributors
 - [Using the testsuite](https://wiki.gpac.io/Build/tests/Using-the-testsuite/): for developpers who want to test their gpac instance
 - [Writing tests](https://wiki.gpac.io/Build/tests/Writing-tests/): technical documentation on writing new tests
